# Video Assets for Creative Events by Apollo

Place the following stock videos in this folder to complete the cinematic website experience:

## Required Videos

| File | Description | Suggested Content | Aspect Ratio |
|------|-------------|-------------------|--------------|
| `hero-main.mp4` | Main hero cinematic video | White floral wedding décor, elegant stage setup, soft celebration moments | 3:4 portrait |
| `hero-wedding.mp4` | Floating wedding card | Intimate wedding moment, bride & groom, floral details | 3:4 portrait |
| `hero-smile.mp4` | Floating smile card | Smiling guests, candid celebration joy | 3:4 portrait |
| `hero-decor.mp4` | Mini decor card + about section | Elegant table setting, flower arrangements, soft lighting | 1:1 square |
| `gallery-moment.mp4` | Gallery video tile | Event highlight reel, celebration b-roll | 1:1 square |

## Video Requirements

- Format: MP4 (H.264)
- Resolution: 720p or 1080p recommended
- Autoplay-ready: muted, no audio required
- Short loops: 8-15 seconds ideal
- Keep file sizes under 5MB for performance

## Where to Get Stock Videos

Recommended sources for soft, elegant, bright event footage:
- Pexels (pexels.com) — free
- Coverr (coverr.co) — free
- Mixkit (mixkit.co) — free
- Artgrid (artgrid.io) — premium cinematic
